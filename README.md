# 💲 Crypto Cotizador 💱
### ◻ _Resumen del Proyecto_
Este proyecto se basa en un cotizador de cryptomonedas, en la que escoges una moneda internacional (Dólar, Euro, Libra y Bolívar), una cryptomoneda (Bitcoin, Ethereum, Dogecoin y Binance Coin) y un monto de la moneda internacional con la que puedas comprar las cryptomonedas. Este proyecto tomó varias inspiraciones de algunas aplicaciones relacionadas con las finanzas como Al Cambio y Rial, pero tomando un estilo minimalista, oscuro y elegante.
